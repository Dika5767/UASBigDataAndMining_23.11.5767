# UASBigDataAndMining_23.11.5767
Repository ini berisi proyek UAS Big Data dan Data Mining dengan topik prediksi emisi CO2 menggunakan dataset publik World Bank. Proyek mencakup preprocessing data, exploratory data analysis, seleksi fitur, modeling regresi linear, dan evaluasi model.  Model dibangun menggunakan Python dan scikit-learn untuk keperluan akademik.
